# Brand Guideline (Minimal for AI)

## 1. Overview（你是誰 + 做什麼）

我們是誰：
→ Taiwan Brands 是一個整理與推廣台灣在地品牌的展示型網站，幫使用者更有系統地認識值得支持的台灣品牌。

我們提供什麼（產品 / 服務）：
→ 蒐集台灣本地品牌資料，整理品牌與產品資訊，讓使用者可以搜尋、探索並找到適合自己的台灣品牌，也讓品牌方有機會被更多人看見。

網站的目標（轉換 / 展示 / 收單）：
→ 以展示與導流為主，幫訪客找到想了解或購買的品牌與產品，進一步主動聯絡品牌；次要目標是吸引品牌方主動提供資訊申請登錄。

---

## 2. Target Audience（你幫誰 + 解什麼問題）

目標使用者：
→ 想購買、認識、支持台灣品牌產品的一般消費者，特別是願意主動尋找有品質、有特色品牌的人。

他們的主要問題：
→ 想支持台灣品牌，但缺少一個可信、整理完整、好瀏覽的地方來快速找到品牌與產品；現有資訊常分散在品牌官網、社群或零散媒體文章中，不易比較與挖掘。

他們想要的結果：
→ 用更省力的方式找到值得關注的台灣品牌，發現符合自己需求與品味的產品，並能直接聯絡品牌或進一步了解。

不服務的對象（可選，但很有用）：
→ 不以大量比價、追求最低價或只想在單一平台直接完成交易的使用者為主要對象。

---

## 3. Core Message（最重要）

一句話價值主張（Tagline）：
→ 在這裡，發現你的台灣在地品牌。

### 為什麼選你（1–2 點）：

- 目前缺少專注整理台灣在地品牌的網站，Taiwan Brands 提供更集中、更完整的探索入口。
- 比起零散媒體介紹或單一品牌官網，這裡更適合用來挖掘品牌、比較風格，並找到下一步聯絡對象。

---

## 4. Tone of Voice（怎麼說）

語氣關鍵字（3 個）：
→ 可信、有品味、好探索

寫作規則：

- 用詞偏好：在地、台灣品牌、精選、探索、發現、整理、值得支持、品牌故事、產品資訊
- 避免用詞：最強、唯一、顛覆、頂級、爆款、便宜、秒殺這類過度行銷或電商促購語氣
- 語氣風格：語氣清楚克制，像有品味且做過功課的品牌策展者；強調整理價值與探索體驗，不過度煽情
