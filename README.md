# Taiwan Brands - 台灣在地品牌

![Taiwan Brands](./public/og-image.png)

Taiwan Brands 是一個整理與推廣台灣在地品牌的展示型網站，蒐集品牌與產品資訊，讓你用更省力的方式探索、發現值得支持的台灣品牌。

## 技術棧

- [Next.js](https://nextjs.org) 16 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript

## 本機開發

```bash
pnpm install
pnpm run dev
```

開啟 [http://localhost:3000](http://localhost:3000) 即可預覽。
